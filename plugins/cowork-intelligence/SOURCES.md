## Notes de rigueur

Pendant la phase de recherche initiale, certaines informations n'ont pas pu être vérifiées contre une source primaire et sont signalées dans les skills :
- Le schéma exact de `marketplace.json` (Anthropic).
- La liste complète des events de hooks au-delà de ceux listés dans la `Hooks reference`.
- Le comportement précis de `disable-model-invocation: true`.
- Les tarifs exacts de prompt caching (1.25× write 5min, 2× write 1h, 0.1× read) — confirmés via plusieurs sources mais à re-vérifier sur la page pricing officielle avant usage contractuel.
- Le beta header exact pour `advanced-tool-use`.
- Les versions exactes de LangGraph / CrewAI / DSPy au moment de la rédaction.

Pour ces points, le plugin utilise systématiquement la formule "verify before relying on this".
